Калькулятор sqf-выражений


Эта очень простая sqf-консоль умеет выполнять выражения и отображать результат в удобоваримом формате. Чтобы выполнить sqf-выражение напечатайте или скопируйте его в поле ввода и нажмите "Enter".

Отформатированный результат вычисления будет отображен в окне "Результат", отредактировать его можно в окне "Буфер". Попробуйте такие примеры:


[weapons player, magazines player]
configFile >> "CfgVehicles" >> typeOf vehicle player
[getpos player nearObjects 100, configFile >> "CfgWeapons" >> "Default"]

Горячие комбинации "Ctrl+C" или "Ctrl+Insert" в окне "Результат" копируют текущую строку результата в окно "Буфер" (в версии под ArmA2 также и буфер обмена), если текущая строка открывает контейнер (открывающая скобка массива или имя класса конфига), то в окно для копирования будет перенесено содержимое всего контейнера.


Каждое выполненное выражение запоминается в истории, двойной клик или нажатие клавиши "Enter" на пункте списка в окне "История" помещает выбранное выражение в поле ввода и выполняет его. Клавиша "Insert" копирует текущий пункт "истории" в поле ввода. Ненужные пункты "истории" можно удалять клавишей "Delete".


Кнопка "Следить" добавляет выражение для наблюдения (будут отображаться в процессе игры в верхнем левом углу), ненужные watcher'ы можно убить в окне "Процессы".


Кнопка-переключатель "Стандартный конфиг" переключает режим отображения конфига,
Стандартный конфиг -- классы показывается как есть, без наследуемых свойств.
Сворачивать конфиг -- отображается только путь в конфиге.
Полный конфиг -- классы отображается со всеми наследуемыми свойствами.


Прочие горячие клавиши:
alt +1 - переключает в поле ввода команд
alt +2 - переключает в окно "Результат"
alt +3 - переключает в окно "Копировать"
alt +4 - переключает в окно "История"
alt +5 - переключает в окно "Процессы"
alt +6 - переключает в окно "Демо"
alt +7 - переключает в окно "Помощь"
alt + стрелка влево - переключиться на соседнее левое окно
alt + стрелка вправо - переключиться на соседнее вправое окно


В папке с вашим профилем вы можете создать подпапку "scripts\SQFCalculator", и разместить там файл "settings" следующего содержания:


// часто используемые вами sqf-выражения,
// эти строки будут добавлены в массив истории
_myHistory = [
'weapons player',
'magazines player'
];

// Коды alt-комбинаций горячих клавиш:
_HKInput = 0x02; // поле ввода sqf-выражения
_HKDisplay = 0x03; // экран с отформатированным результатом
_HKDisplay2 = 0x04; // экран редактирования результата
_HKHistory = 0x05; // экран редактирования результата
_HKDemo = 0x06; // экран html-демок (скриптов, команд, итк)
_HKHelp = 0x07; // экран помощи
_HKLeft = 0xCB; // переключить на соседний левый экран
_HKRight = 0xCD; // переключить на соседний правый экран
