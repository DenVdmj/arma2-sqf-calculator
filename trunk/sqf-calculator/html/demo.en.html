Demo


For using the demo-mode (only in your mission), you need to add to description.ext the following class:

class RscSqfCalcDemo {
htmlfile = "someDemo.html";
htmlControlHeight = 3;
};

and to write someDemo.html. The format is the same as that of help.en.html. You can also use sqf-links:

<a href="sqf:hint &quot;this simple sqf-link&quot;"
>hint &quot;this simple sqf-link&quot;</a>

It would look and work like this:

hint "this simple sqf-link"
